Custom Javascript Views

A framework for custom views that enhance the DOM from javascript.

Custom views are defined from script using a unique name and descriptor object as follows: 

Slender.defineView("Slender.ViewName", {

	// Override
	onEnhance: function () {
		// Enhance the DOM from this method, this.element is a jQuery object
		this._oldContent = this.element.html();
		this.element.text("Enhanced");
	},

	// Override
	onRestore: function () {
		// Restore the DOM from this method
		this.element.html(this._oldContent);
	}

});

The view Slender.TestView is always available. It sets the elements' text to "Enhanced" and "Restored". It is shown below and is only used for testing. To activate the custom view framework, add a data-view="Namespace.Name" attribute to an element and the framework will do the rest.
