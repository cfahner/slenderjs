.sl-mwidth

The class sl-mwidth applies a default site-wide maximum content width to a block element. This prevents content being stretched to 100% width on large screens.

.sl-mid

The class sl-mid adds margin:0 auto; to an element and therefore centers block elements.

.sl-grid

Grids display a number of elements side by side. Add the class sl-grid to an element to create a grid. By default the columns of a grid take up the full width of the grid. To get custom column widths, add one of the sl-w-* classes to the column.

Columns cannot be styled directly. Add an inner container element and style that instead (padding, margins, borders).

All grids stack when on small screens.

.sl-clear

To clear a row in a grid, add an sl-clear element to the grid at the point that should break the row on large screens.

.sl-floatr, .sl-floatl

Float element easily using sl-floatr (right) or sl-floatl (left). Use sl-clear to clear the floating elements.
