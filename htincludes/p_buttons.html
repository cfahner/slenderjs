Basic buttons

Enhance elements to appear like a button by adding the CSS class sl-btn to it. For color input types, you should add a style attribute defining a min-width (style="min-width:50px;") to prevent collapse of it's width to near invisibility.

Action buttons

Further enhance buttons by adding action classes to them, indicating these buttons perform special actions by adding a color.

.sl-action-do

Add the sl-action-do class to buttons that immediately perform an action without asking for confirmation (or are confirmation buttons themselves).

Link button

.sl-action-danger

Add the sl-action-danger class to buttons to indicate that this button performs a dangerous action (such as permanent removal of an item).

Flat buttons

Buttons can also be 'flat'. These buttons do not have any styling other than the default button margins and padding. Use sl-flat to create a flat button.

Link

Fullwidth buttons

Buttons can forced to 100% width by using the sl-w-fill class.

A class called sl-w-fill-mobile will make buttons stretch only on small screens. This will make multiple buttons appear next to each other on medium-large screens and stacked on small screens.

Unstyled buttons

Not adding the sl-btn class will keep their default style.
