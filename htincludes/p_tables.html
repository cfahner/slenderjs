Tables

There is no good CSS only "fix" for automagically responsive tables. The closest to an "automagic" solution I could find was this: css-tricks.com/responsive-data-tables.

An attempt

This attempt requires custom CSS to set the content of the ":before" pseudo element to the column heading text. In this example:

@media (max-width: 65em) {
	#MyCustomTable td:nth-of-type(1):before { content: "First name"; }
	#MyCustomTable td:nth-of-type(2):before { content: "Last name"; }
	#MyCustomTable td:nth-of-type(3):before { content: "Gender"; }
}
