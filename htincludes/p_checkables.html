Radio groups

Enhance radio-buttons by adding the sl-radio class to the input type=radio and the sl-lbl class to the label right after it.

Source of radio button group below: 

<input class=sl-radio id=radio1 type=radio name=radios value=1 checked>
<label class=sl-lbl for=radio1>Radio 1</label>
<input class=sl-radio id=radio2 type=radio name=radios value=2>
<label class=sl-lbl for=radio2>Radio 2</label>
<input class=sl-radio id=radio3 type=radio name=radios value=3 disabled>
<label class=sl-lbl for=radio3>Radio 3</label>

Checkboxes

Checkboxes work the same as radio buttons, but they use the class sl-checkbox instead of sl-radio.

Switches

Switches are <select> input's with only two options. They are only enhanced to switches from script.

<select class=sl-switch>
  <option class=sl-switch-off value=0>Off option</option>
  <option class=sl-switch-on value=1>On option</option>
</select>
