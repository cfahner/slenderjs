Styling block elements

The following classes can be used for styling block level elements.

.sl-padding

The class sl-padding only adds a default padding to an element.

.sl-margins

The class sl-margins only adds default margins to an element.

.sl-padding .sl-margins

These classes can be combined if needed.

.sl-block-styled

The class sl-block-styled only adds default block level styling to an element (non layout related). This includes background, borders and shadows.

.sl-block

The class sl-block is a short combined class for all classes defined above.
