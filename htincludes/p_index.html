Text special classes

The following classes allow easy text changes (prefixed with sl-t...).

sl-tmid (align text center)

sl-tleft (align text left)

sl-tright (align text right)

sl-tsmaller (smaller text size)

sl-tlarger (larger text size)

sl-tmono (monospace font)

sl-tstart (start aligned, use dir attr)

sl-tend (end aligned, use dir attr)

sl-tmid-mobile (center aligned on mobile only)
