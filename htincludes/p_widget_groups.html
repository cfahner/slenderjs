Horizontal widget groups

Create horizontally grouped widgets by wrapping a number of widgets with the sl-widgetgroup-h class.

Use the sl-w-* classes to specify the width of each widget in the group.

The sl-w-auto class can also be added to widgetgroups.

Vertical widget groups

Create a vertical widgetgroup using the sl-widgetgroup-v class.

Vertical radio widget group.

Nested widget groups
