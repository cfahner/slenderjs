.sl-nojs

Elements with the sl-nojs class are removed when javascript is enabled (and stay visible if disabled).

The delaytextchange event

All elements that match input, select, textarea receive a delaytextchange event after the user changes the value of the input and a certain (typing) delay has expired. When another change happens in between that delay, the delay will be renewed.

.sl-autosubmit and .sl-autosubmit-delay

The .sl-autosubmit class can be used to automatically submit the closest form. When the change event is triggered on these elements, they will automatically submit that form.

The .sl-autosubmit-delay class does the same thing, except this class relies upon the delaytextchange event described above.
